My workflow

To make, host and share my projects I use Replit. I am using to even host this website! I use it for multiple reason. First off, I adamantly support their mission to level the playing field when it comes to software development, bringing an entire IDE, VM and frameworks to the web in a lightweight way, enabling everyone with a device to code. Second, it is really convenient to create projects in it, since you have thousands of templates to choose from in Replit which come pre-configured, and with its Nix infrastructure, you can install basically anything to your project. Lastly, each project has a professional IDE, version control, debugger, database and package manager. As a bonus, each project is really easy to share, as you can just send the project link. If I managed to interest you in Replit, check it out here.
